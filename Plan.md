# Plan - Cycle 4: 퍼펙트 게임 (올스트라이크, 확인 테스트)

## 목표
12번 연속 스트라이크(투구 10을 12번) 시 `score()`가 300을 반환하는지 확인.
프레임9까지 스트라이크 체인 + 10번 프레임 스트라이크 + 보너스 2회(마지막 2번의 10)가
현재 스트라이크 로직(각 프레임 `i += 1`)으로 이미 올바르게 처리되는지 검증하는 확인 테스트.

## 테스트
- 파일: `test_game.py`
- `test_perfect_game`: `roll(10)`을 12번 호출 후 `score() == 300`

## 접근 방식

- 통과하면: 확인 테스트로 커밋 (일반화 완료 확인, 코드 변경 없음)
