# Plan - Cycle 2: 스페어 보너스

## 목표
한 프레임에서 스페어(두 투구 합 10) 발생 시, 해당 프레임 점수에 다음 투구 1회의 핀 수를 보너스로 더한다.

## 테스트
- 파일: `test_game.py`
- `test_spare_bonus`: 첫 프레임 5,5(스페어) + 다음 투구 3 + 나머지 17회 거터(0)
  - 기대 점수: (10+3) + 3 + 0*16 = 16

## 접근 방식
- `score()`를 프레임 단위 순회로 재작성
- 각 프레임 시작 인덱스에서 두 투구 합이 10이면 스페어 처리: `10 + rolls[다음 인덱스]`를 프레임 점수로 더하고 인덱스 2칸 이동
- 스페어 아니면 두 투구 합을 그대로 더하고 인덱스 2칸 이동
- 프레임 10개만 순회 (10번 프레임 보너스 투구는 다음 사이클)

## 범위 밖
- 스트라이크 보너스 (다음 사이클)
- 10번 프레임 보너스 투구 (다음 사이클)
- 입력 검증
